# AI News Aggregation + Summarization Automation

## Overview
This project automates the process of collecting, summarizing, and delivering news updates using AI-powered workflows.

The workflow processes news content, generates concise summaries, and sends automated notifications through email.

## Features
- Automated news aggregation
- AI-generated news summarization
- Automated email notifications
- Workflow automation using n8n
- Structured information delivery

## Technologies Used
- n8n
- AI APIs
- Email Automation
- Workflow Automation

## Workflow
1. Collect news/input data
2. Process content using AI
3. Generate summarized output
4. Send automated email notifications

## Files
- `workflow.json` → exported n8n workflow
- `automated_email_output.png` → sample automated email output
